
# Threat Intelligence Analyzer

A Flask-based Threat Intelligence platform for analyzing PDF security reports.

## Features

- PDF Upload
- IOC Extraction
- Risk Scoring
- SQLite Database
- Analysis History
- Security Dashboard

## Technologies

- Python
- Flask
- SQLite
- PyMuPDF
- Bootstrap
